## 1장. 핵심기능

### 사전 준비 및 추진 계획

* 게임 설계  
  ` 게임 시작화면에서 시작 버튼을 눌러 게임화면에 입장 `,
  ` 시작하자 마자 시간이 가면서 플레이어는 카드 짝을 맞춤 `
  ` 맞추기 성공 시 해당 카드의 주인과 설명을 잠시동안 보여줌 `
  ` 성공하면 모두의 TMI를 볼 수 있고, 돌아가기 버튼이 존재하는 성공창을 띄움 `
  ` 실패하면 다시하기와 돌아가기 버튼이 존재하는 실패창을 띄움 `
